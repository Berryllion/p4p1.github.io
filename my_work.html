p4p1 | get here

It is a remote administration tool that I built to control Windows computers via the command line from a Linux
system. The server sits on a Linux box waiting for clients to connect.

When a client connects successfully you can then list the clients that are connected and then either send a
command to all of them at once or initiate a shell with one an other

larp | get here

Also called Leo's ARP is a arpspoofing software. It sends ARP packets to the gateway and to all of the clients on the network. This alows you to filter all the packets on the network.

Curently larp has a shell interface so that you can execute sniffer with the rules you wish to use on them. It will also include in the future more options like a stealth mode and an automatic config using cfg files.

p4p1 USB Loader

USB stick to load p4p1.exe from an online repository to the machine it's plugged in. I used a digispark microcontroller with the USB keyboard library to use it as a keystroke injection tool.

The microcontroller starts by opening the Windows Start Menu, it waits then 5 seconds before entering a big power shell command that will install the program in the startup folder and then it will reboot the machine, after agreeing to UAC.

Rpi USB to Ethernet gadget

This Raspberry Pi Zero, has a Ethernet port soldered on to it, so that it can share it's network to the computer it's plugged in

You can connect to the Raspberry pi Zero using ssh via the USB port or the Ethernet one. The USB provides the data and the power to the pi and the ethernet the network. This gadget could be used to do some packet sniffing on a local network or just as a cheap ethernet to usb adaptator.
